## 1. 游戏概述

### 1.1 核心概念

《魔神战争:重现》是一款 **3D 多人乱斗竞技游戏**，核心体验来自 **卡牌轮转机制** 与 **元素反应系统** 的融合。游戏试图回答一个问题：如果把《原神》的元素交互逻辑放进《皇室战争》的卡牌循环框架，并且扔进一个快节奏的死亡竞赛场，会变成什么？

### 1.2 体验目标

- **30秒学会，300小时精通**：基础操作极简（只有移动+4个技能键），但元素反应、地形利用、卡序预判的上限极高。
- **每局都是新战场**：冰晶生长、活石转化、门扉移动等机制让竞技场永远在变化。
- **决策压力来自“现在出哪张”**：魔力有限，技能会轮转，打错一张牌的惩罚不是损失资源，而是失去战机。

### 1.3 参考对标

| 游戏         | 借鉴要素               |
| ---------- | ------------------ |
| 皇室战争       | 魔力自动回复、卡牌轮转、卡组构筑   |
| 原神         | 元素附着、元素反应、元素类型克制逻辑 |
| Spellbreak | 元素组合、3D环境中的法术对战    |

### 1.4 目标平台与性能指标

- 首发 PC（键鼠适配），移动端延后适配。
- 帧率：PC端稳定60fps，竞技场最大同时在线玩家8人，技能特效密集时需保证不下40fps。
- 网络：采用帧同步或状态同步方案，需支持弱网下的技能命中判定（客户端预测+服务端验证）。

## 2. 核心玩法循环

### 2.1 宏观循环

一场对局分为三个阶段：

- **开局（0-60秒）**：玩家落地，魔力初始0点，迅速回复。这时以低费技能试探为主，少量地形争夺（抢占活石附近位置）。碰撞不激烈，核心任务是“把卡序转到你想要的起手”。
- **中期（60-120秒）**：魔力充裕，大技能开始释放。冰晶地形初步成型，活石被触发转化，门扉出现。这是淘汰率最高的阶段，地形复杂度最高。
- **终局（120秒后或剩余3人时）**：场地边缘开始崩塌（可选机制），魔力回复速度加倍（或减半，取决于设计方向）。剩余玩家被迫贴身交战，生存技能价值飙升。

### 2.2 微观循环（技能释放决策树）

玩家每时每刻都在回答三个问题：

1. **当前手牌中哪张最有用？**——考虑敌人距离、地形、自身血量。
2. **打出这张牌后，下一张轮转上来的是什么？**——打火球是为下一张冰墙腾位，还是会把关键净化洗走？
3. **我这手会触发什么连锁反应？**——冰球砸在地面生成冰晶，下一张水雨会让整个区域冻结。对手的活石就在旁边，打雷会磁化全场。

## 4. 核心机制：魔力与卡牌轮转

### 4.1 魔力系统

- **上限**：10点。
- **回复速度**：基础1.0点/秒，使用“侵染”技能后可提升至2.0点/秒。
- **起始魔力**：0点（鼓励开局低费试探）。
- **溢出惩罚**：满魔力后不再回复，迫使玩家必须消费魔力。
- **击杀奖励**：击杀敌方玩家立即回复2点魔力（鼓励进攻）。

### 4.2 卡组构筑（赛前）

每位玩家在进入对局前需完成以下步骤：

1. **技能解锁**：玩家拟通过游戏进程（等级/任务）获得技能。在测试开发阶段默认全解锁。
2. **构筑卡组**：从已解锁技能中选择8个组成卡组。
3. **构筑限制**：不能全部选择同一元素（防止无脑堆砌），建议至少包含2个元素类型。系统对元素分布不做强制限制，但会提示“元素过于单一可能被克制”。
4. **排序机制**：8张牌的顺序在开局时随机排列，玩家无法手动排序。这意味着你需要熟悉自己的8张牌，并在战斗中实时调整策略——高端玩家会“算牌”，知道接下来3-4张是什么。

### 4.3 轮转机制

初始卡序：[火球, 水疗, 冰球, 风刃, 火波, 导电, 岩种, 缠绕]
当前手牌：[火球, 水疗, 冰球, 风刃] ← 4个槽位

打出火球后：
当前手牌：[水疗, 冰球, 风刃, 火波] ← 火波轮转上来
卡序队列：[导电, 岩种, 缠绕, 火球] ← 火球进入队尾

## 5. 元素系统

### 5.1 元素哲学



月矩力与虚界力是“双刃剑”设计——高风险高回报。

### 5.2 元素附着机制

- **附着值**：每个提供附着的技能有隐藏的“附着量”。1次附着=1单位，触发一次反应消耗1单位。
- **衰减**：附着在目标上的元素持续3-5秒后自然消失（取决于元素类型，火附着衰减最快）。
- **附着叠加**：同一目标可同时存在多种元素附着，但一旦触发反应，对应的两种元素附着同时消耗。
- **附着视觉反馈**：玩家角色身上出现对应元素光效——火焰缠绕、水滴包裹、冰霜覆盖、电光闪烁等。

**冰晶属性**：

- 生成逻辑：冰元素攻击可使场景进入**冰冻**状态，此时加以水元素攻击即可生成冰晶。
- 生命值：3次中等攻击可摧毁（火元素攻击双倍伤害）。
- 交互体积：1m×1m×2m，可站立、可阻挡投射物。
- 增殖条件：水雨或水球落在冰晶上时，冰晶会增长变大（最多增长3次），并对自身范围造成冰元素范围伤害。
- 融化：火元素攻击直接摧毁冰晶，并在3秒内对该冰晶所在区域附加“湿滑”效果（减速）。

## 7. 元素反应与特殊交互

### 7.1 反应表

| 元素组合   | 反应名 | 效果                  |
| ------ | --- | ------------------- |
| 火 + 水  | 蒸发  | 额外伤害+蒸汽烟雾（短暂视野遮挡）   |
| 火 + 冰  | 融化  | 冰晶熔化，额外伤害，生成湿滑地面    |
| 水 + 雷  | 感电  | 连锁伤害，传导至周围水附着单位     |
| 火 + 雷  | 超载  | 范围爆炸+弹飞             |
| 冰 + 雷  | 超导  | 目标防御力降低30%，持续4秒     |
| 风 + 任意 | 扩散  | 将单体附着扩散至周围          |
| 草 + 火  | 燃烧  | 区域持续火伤，草果陷阱被点燃后伤害翻倍 |
| 水 + 冰  | 冻结  | 冰晶增殖/目标短暂冻结（若为实体目标） |

### 7.2 特殊交互

- **水雨 + 冰晶 → 增殖**：水雨区域内，每块冰晶可生成2块次级冰晶，次级冰晶不可再次增殖。
- **水雨 + 岩浆区 → 黑曜石**：火+岩的熔岩区域遇到水雨会凝固为黑曜石（可站立的临时平台，不可摧毁）。
- **雷光 + 磁化目标 → 必定命中**：磁化单位对雷系技能有“吸引”效果，雷光蓄力时若有磁化目标在范围内，释放后将自动导向该目标。

## 8. 战场设计

### 8.1 基础地形

- **主要平台**：直径约80米的圆形浮动岛屿，边缘即使虚空。
- **次级平台**：3-4个浮空小岛，通过气流或跳跃可到达，提供远程狙击位。
- **冰晶初始为0**，完全由玩家技能生成。

### 8.2 动态事件

- **60秒事件**：“元素潮汐”——随机一种元素伤害+20%，持续15秒。
- **90秒事件**：“深渊裂隙”——场地边缘开始崩塌，战斗区域缩小20%。
- **120秒事件**：“魔力狂潮”——魔力回复速度翻倍，持续至对局结束。

### 8.3 深渊规则

- 坠落虚空：立即淘汰。
- 月引（免疫摔落伤害）：在持续时间内若坠落深渊，自动传送回最近的安全地表。
- 月斥（向上弹飞）：高度超过15米后受重力影响加速下落，若落点无平台则直接淘汰。

## 9.游玩模式设计

### 9.1 单挑模式
一场游戏设置n个回合，在一个回合内，每位玩家只有一条命，最终存活玩家获得该回合胜利。
### 9.2 团队模式
一场游戏设置两个团队，玩家可以复活，最终击杀数最多的团队获胜
